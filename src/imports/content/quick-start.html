Step 1

Download the h2zero jar file

Download the required dependencies

Step 2

Create your .h2zero file

To get started quickly, you can use the following h2zero file as a reference point - which is used throughout the examples listed in this documentation set.

This file generates a simple schema for a database that records users and their pets. Each user may have multiple pets and each pet may have multiple users. 

	
{import src/imports/h2zero/sample.templar}

Step 3

Set up your ant build.xml file 

	
<?xml version="1.0" encoding="UTF-8"?>{\n}
<project name="h2zero-sample">{\n}
{\t}<path id="classpath-h2zero">{\n}
{\t}{\t}<fileset dir="lib/runtime">{\n}
{\t}{\t}{\t}<include name="*.jar"/>{\n}
{\t}{\t}</fileset>{\n}
{\t}</path>{\n}
{\n}
{\t}<target name="h2zero" description="h2zero generate">{\n}
{\t}{\t}<taskdef name="h2zero" classpathref="classpath-h2zero" classname="synapticloop.h2zero.ant.H2ZeroTask" />{\n}
{\t}{\t}<h2zero in="src/main/java/sample.h2zero" outDir="." verbose="false" />{\n}
{\t}</target>{\n}
{\n}
</project>{\n}

Step 4

Generate the code! 

	
ant h2zero

This will generate all of the required code for you h2zero file and database creation scripts, see the {import src/imports/links/generated-code.templar} page for an overview of the directory structure and the code that is generated.
