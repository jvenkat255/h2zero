The h2zero file format.

The h2zero file format is a JSON formatted document with a specific layout as per below. The various sections are highlighted and linked to thedocumentation for that key.

{
	"options": {
		...
	},
	"database": {
		"schema": "...",
		"package": "...",
		"tables": [
			{
				"name": "...",
				"fields": [
					{...}
				],
				"finders": [
					{...}
				],
				"fieldFinders": [
					{...}
				],
				"deleters": [
					{...}
				],
				"inserters": [
					{...}
				],
				"updaters": [
					{...}
				],
				"questions": [
					{...}
				],
				"counters": [
					{...}
				],
				"constants": [
					{...}
				]
			}
		], 
		"views": [
			{...}
		]
	}
}

The navigation follows the file format, and you may click on any of the JSON keys above to get more information from the relevant section.

See the Quick Start page for more a more detailed sample/example .h2zero file.
